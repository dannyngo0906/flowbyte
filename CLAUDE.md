# Tài liệu dự án
- PRD (yêu cầu sản phẩm): @documents/PRD-flowbyte.md 
- TDD (thiết kế kỹ thuật): @documents/TDD-flowbyte.md 

# Nguyên tắc khi implement
- Đọc User Story trong PRD, implement đầy đủ theo từng tiêu chí "Done khi"
- Tham chiếu TDD cho mọi quyết định kỹ thuật: tech stack, DB schema, API design
- Hỏi trước khi làm nếu có gì chưa rõ trong PRD hoặc TDD
- DO NOT xóa các file PRD và TDD.
